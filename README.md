# Dev Yatri-journey to software engineer.

[![Live Demo](https://img.shields.io/badge/demo-online-brightgreen)](https://devyatri-blog-app.onrender.com)

![Dev Yatri Demo](https://via.placeholder.com/1200x630.png?text=Dev+Yatri+Demo)

A clean, minimal developer blog built with React (Vite) and a small Node backend. "Dev Yatri" (Dev Traveler) showcases posts, projects, and personal journey notes.

**Demo**: https://devyatri-blog-app.onrender.com

**Screenshot / Hero image**: <img width="1898" height="932" alt="image" src="https://github.com/user-attachments/assets/1b54d61e-5a94-4100-b7e0-dcd8bbd69607" />

## Features

- Lightweight Vite + React frontend
- Simple Node backend (API for posts)
- Responsive design with Tailwind CSS
- Reusable components: `Navbar`, `PostCard`, `Newsletter`, etc.

## Tech Stack

- Frontend: React, Vite, Tailwind CSS
- Backend: Node.js (Express or similar)
- Deployment: Vercel / Netlify (suggested)

## Project structure

- frontend/ — React app (Vite)
- backend/ — Node API and server

## Getting started (local)

Prerequisites: Node.js 14+ and npm

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Backend (if applicable):

```bash
cd backend
npm install
npm run start
```

Build frontend for production:

```bash
cd frontend
npm run build
```

## Deploying a demo

- Recommended: Deploy the `frontend` directory to render
